Paragraf

TML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Dengan HTML, kita bisa membuat struktur dasar sebuah halaman web menggunakan berbagai elemen seperti heading, paragraf, gambar, dan lainnya. Setiap elemen HTML ditandai dengan tag pembuka dan tag penutup.

Contoh paragraf ini menunjukkan bagaimana teks panjang bisa ditampilkan dalam halaman web. Paragraf sangat penting untuk menyajikan informasi secara terstruktur dan mudah dibaca.

Daftar Berurut

  1. Memahami struktur dasar dokumen HTML
  2. Mempelajari tag-tag dasar seperti html, head, body dan title
  3. Menggunakan elemen-elemen teks seperti heading dan paragraf
  4. Menambahkan gambar ke halaman web
  5. Membuat link dengan elemen a href
  6. Membuat daftar dengan ul dan ol
